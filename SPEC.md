# ボードゲーム印刷 見積もりツール — 要件定義

## 目的

萬印堂（https://www.mnd.co.jp/pricelists/）の価格表に基づき、ボードゲームの印刷費用をブラウザ上で見積もれるツール。ビルド不要・サーバー不要で動作し、URLで見積もり内容を共有できる。

## 技術スタック

| 項目 | 内容 |
|------|------|
| フロントエンド | HTML / CSS / JavaScript（ESモジュール） |
| UIフレームワーク | Vue 3（CDN経由、ビルドなし） |
| サーバー | 不要。ローカル確認は `_server/`（ASP.NET Core Kestrel）を使用 |
| 外部依存 | なし（Vue CDN以外） |

## 機能要件

### F1 商品フォーム
- 商品種別をボタンで選択し、種別ごとのフィールド（サイズ・加工・厚さ・枚数など）が動的に表示される
- 商品を選択すると価格プレビューがリアルタイムで更新される
- 「見積もりに追加」で見積もり一覧にアイテムを追加する

### F2 商品・価格データ
- 対象商品: カード / ゲームチップ / ゲームボード / 化粧箱 / キャラメル箱 / プレイマット / プレイシート / 説明書 / 特注ダイス / ペーパールーレット / プレイヤーコマ・ゲーム用ついたて / 木製品・樹脂製品・その他既製品
- **商品名・サイズ名は価格表ページの正式表記に完全準拠する**（独自の言い換え禁止）
- 価格表で別サイズとして列挙されているものはすべて選択肢に含める
- 価格は税抜きで計算し、税込（10%）は参考表示

### F3 ゲーム製造個数
- 全体共通の「製造するゲームの個数」（初期値 50）を設ける
- 各商品の個数入力に「1ゲームあたり」「合計個数」のモード切替を設ける（デフォルト: 1ゲームあたり）
- 「1ゲームあたり」の場合: 実製造数 = 入力値 × ゲーム個数
- ゲーム個数を変更すると、追加済みアイテムの金額がリアルタイムで追従する

### F4 見積もり一覧
- 各アイテムにスペック・計算式（基本料 + 数量 × 単価）を表示する
- 合計（税抜）・合計（税込）・1ゲームあたり単価（税抜）を表示する
- アイテムを個別に削除できる

### F5 URL共有
- 見積もりの全状態（ゲーム個数・各アイテムのスペック・数量モード）を URLパラメータ（`?data=`）に保存する
- URLを開くと状態が復元される

## 非機能要件

- **レスポンシブ**: 720px 未満は1カラム、720px 以上は2カラム
- **アクセシビリティ**: 操作しやすさ・見やすさを装飾より優先する
- **免責表示**: 価格が古い可能性・誤りを含む旨をページ上に明示し、公式価格表へのリンクを設ける

## スコープ外

- バックエンド・データベース・認証
- 正式な発注機能（見積もりの計算のみ）
- オフライン動作・PWA
